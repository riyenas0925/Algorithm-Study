## STL sort
* default값은 오름차순

## 배열일 경우
```c++
int a[100];
// sort(배열의 포인터, 배열의 포인터 + 배열의 크기)
sort(a, a+100)
```

## vector일 경우
```c++
vector<int> a(100);
// sort(iterator의 begin(), iterator의 end())
sort(a.begin(), a.end());

sort(a.begin(), a.end(), greater<int>());   // 내림차순
sort(a.begin(), a.end(), less<int>());      // 오름차순 (default);
```
